## 一、核心策略与创新点

### 1.1 研究背景与核心思想
研报研究「股债跷跷板」效应——股票与债券价格反向联动，并将其转化为国债期货（及中债现券指数）的择时信号；再以此效应改进机构前作的「隔日反转」国债期货择时策略。核心思想：股市的区间/单日/周内表现对下一期国债走向具方向性预测力，叠加隔日反转因子构成复合择时。

### 1.2 因子/信号/策略构造（关键公式，细节以 spec.md F1–F8 为准）
1. 长短端跷跷板（F1）：chg_T^N = 沪深300过去N个交易日涨跌幅；Upper/Lower_Bound = 基日(2004-12-31)以来 chg_t^N 的上/下 1/3 分位；chg_T^N>Upper→牛市→T日收盘做空国债期货，<Lower→熊市→做多，区间内→平仓；N_l=120、N_s=20；信号 signal_l_s = sign(长端+短端)。每日操作。
2. 日度信号（F2）：股市单日涨跌幅绝对值触发高/低阈值 l_upper/l_lower（AS1 采用±5%/±3%）→ 下一交易日反方向操作国债期货。
3. 隔日反转因子（F3，公式由 **AS4/human 裁决 A1** 补齐）：chg_t = 国债期货主力合约 T-2 日结算价涨跌幅 = settle_{T-2}/settle_{T-3} − 1；当 |chg_t|∈[0.03%,0.5%]（含边界）时反转信号激活：chg_t>0→做空(-1)、chg_t<0→做多(+1)，信号 T 日执行；区间外（|chg_t|<0.03% 或 >0.5%）信号为 0。公式源自前作复原材料（见 AS4/歧义 A1），coder 从公式独立实现。
4. 复合信号（F4）：signal_seesaw = sign(signal_daily_lower + signal_l_s)。
5. 周内信号（F5）：signal_calendar = 1 if 周四 else 0；signal_daily_upper_calendar = sign(signal_daily_upper + signal_calendar)。
6. 决策树策略（F6/F7/F8）：F6 分段（区间内用隔日反转，超区间用跷跷板，>2%处理见AS3）；F7 三步（signal_daily_upper 触发→隔日反转→signal_seesaw）；F8 在 F7 首步换入周内信号，为最终策略。

### 1.3 创新点
相对前作单一隔日反转策略：逐层引入长短端跷跷板、日度/复合跷跷板、周内效应，构成三步决策树复合择时。最终 F8 将区间收益从隔日反转的47.82%（R7）提升至85.22%、年化7.65%、夏普2.04（R11）。

## 五、改进建议与潜在问题（AI 视角）
1. **A1 已由 human 裁决解锁（原最高优先缺口）**：chg_t 是全报告后半基石，已由人工闸门提供前作《国债期货的"隔日反转"效应及择时策略构建》(2023-05-13) 公式复原材料补齐（AS4）。实现时优先落地 m4 的 AS4 公式口径并以 R7 反向校验；R7 是前作公式复原可信度的试金石——若 R7 超差且归因指向 chg_t，说明复原公式与研报实际口径有偏差，需 revise 回看（结算价字段、区间边界、执行 lag）。仍建议先交付 m1–m2（跷跷板前半线，不依赖 chg_t；m3 现券已裁决 infeasible 移除），再依序推进 m4–m8。
2. **信号时点对齐风险**：国债期货收盘(15:15)晚于A股(15:00)15分钟，跷跷板信号（沪深300 T日收盘）在 T 日国债期货收盘可执行，无未来函数；但隔日反转 chg_t 用 T-2 日结算价判 T 日方向，须确认执行日与 lag，避免 look-ahead。
3. **日度阈值敏感性（AS1）**：±5%/±3% 为研报正文策略定义处的示例运行值；表2/3/4（R2/R3/R4，SA-A01 补录）为日度效应存在性证据表而非策略参数声明，补录后可作 ±5%/±3% 档位合理性的旁证（对照 15 档效应强度）。若 R9/R11 复现超差且归因指向阈值，应参照表2/3/4 的 15 档效应强度在 revise 时切换较优档位重测。
4. **现券代码覆盖（m3）已裁决 infeasible**：经穷尽复核（audit/m3_data_infeasibility.md，planner 独立重算坐实），4 个 CBA 代码及等价中债净价指数在本地全部债券表零行情覆盖（中央结算 5509 中债指数与 quote 行情内码零交集），R5/R6 无本地数据支撑——m3 已移除、进入最终报告未复现清单（见 AS5/A5）；国债期货主线（含落脚点 R11）不受影响。
5. **vwap_X 数据缺口（SA2/R13）**：本地无分钟数据，成交价敏感性只能降级复现；因 SA2/R13/B2 均为 support 级，不影响主结论（收盘价口径的 R11）。
