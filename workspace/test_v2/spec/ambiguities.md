### [A1] 隔日反转因子 chg_t 计算公式缺失

- 关联要素: F3, F6, F7, F8, R7, R8, R9, R11, R14
- 等级: blocking
- 候选解释: 1. 参照前作《国债期货的"隔日反转"效应及择时策略构建》（2023-05-13 发布）复原 chg_t 完整公式 2. 按"T-2 日国债期货结算价计算的修正反转因子"通用定义做假设实现，以能否复现 R7（隔日反转多空年化收益 4.78%）反向校验
- 裁决方式: human（人工闸门裁决——planner 曾判 blocking/blocked 暴露缺口交主流程暂停，用户在闸门处选择采用人工提供的前作公式复原材料，非 planner auto 降级）
- 裁决结果: 采用人工提供的前作公式复原材料：隔日反转因子 chg_t = 国债期货主力合约 T-2 日结算价涨跌幅（settle_{T-2}/settle_{T-3} − 1）；当 |chg_t| ∈ [0.03%, 0.5%]（含边界）时反转信号激活：chg_t > 0 → 做空（−1），chg_t < 0 → 做多（+1）；信号在 T 日执行；区间外信号为 0。登记 AS4（human 裁决核心公式，高亮按 major-auto 级同等对待，强制进 final_report 假设章节，verify 后回看是否被 R7 数据支持）；feasibility 随之由 blocked 复评为 degraded，解锁 m4 及下游 m5–m8
- 裁决依据: 人工提供的前作《国债期货的"隔日反转"效应及择时策略构建》(2023-05-13) 公式复原材料——旧系统复现同一研报时从前作复原 chg_t 完整公式，实现于 src/test/strategy.py:187-209（calculate_reverse_signal），当年经验证多空年化偏差 <2.5%（见 output/test/verify_report.md）；该材料仅作 chg_t 公式定义来源，禁止在本次 plan 引用其代码文件路径作为实现依赖，新实现由 coder 从公式独立编写。人工闸门补齐后 spec F3「本报告未给出完整计算式」的缺口消解，A1 由 blocked 转 resolved
- 状态: resolved
