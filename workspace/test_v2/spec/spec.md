## 三、因子/策略定义（F 类）

### [F1] 长短端股债跷跷板国债期货择时策略
- 页码: p6-p7
- 原文: > "定义𝑐ℎ𝑔𝑡𝑁 的上轨（Upper_Bound）和下轨（Lower_Bound）分别为沪深 300 基日以来至𝑇日，𝑐ℎ𝑔𝑡𝑁 的上三分之一分位点和下三分之一分位点……长端策略中𝑁𝑙=120（6 个月），短端策略中𝑁𝑠=20（1 个月）"
- 公式（转写）: chg_T^N = 沪深300 过去 N 个交易日涨跌幅；Upper_Bound = 基日以来 chg_t^N 的上 1/3 分位；Lower_Bound = 下 1/3 分位。若 chg_T^N > Upper_Bound → 牛市 → T 日收盘做空国债期货；若 chg_T^N < Lower_Bound → 熊市 → T 日收盘做多；若 chg_T^N ∈ [Lower_Bound, Upper_Bound] → 平仓；每日操作。长短端策略信号 = sign(长端信号 + 短端信号)
- 参数: 长端 N_l=120；短端 N_s=20；分位阈值=上/下 1/3；执行=T 日收盘
- 依赖数据: D1, D2 / 关联基准: R1, R5, R6 / 关联歧义: （无）

### [F2] 日度级别股债跷跷板信号（signal_daily）
- 页码: p13
- 原文: > "𝑠𝑖𝑔𝑛𝑎𝑙𝑑𝑎𝑖𝑙𝑦_𝑢𝑝𝑝𝑒𝑟 为股市单日涨跌幅触发较高的阈值𝑙𝑢𝑝𝑝𝑒𝑟（如±5%）时的择时信号，𝑠𝑖𝑔𝑛𝑎𝑙𝑑𝑎𝑖𝑙𝑦_𝑙𝑜𝑤𝑒𝑟 为股市单日涨跌幅触发较低的阈值𝑙𝑙𝑜𝑤𝑒𝑟（如±3%）时的择时信号"
- 公式（转写）: 考察股市单日涨跌幅与国债期货下一交易日涨跌幅的反向关系；股市单日涨跌幅绝对值触发阈值 → 下一交易日反方向操作国债期货（大涨→下日做空，大跌→下日做多）
- 参数: l_upper 高阈值（研报示例 ±5%）；l_lower 低阈值（研报示例 ±3%）
- 依赖数据: D1, D2 / 关联基准: R2, R3, R4 / 关联歧义: A2, A3

### [F3] 隔日反转因子 chg_t（前作回顾）
- 页码: p11-p12
- 原文: > "通过𝑻−𝟐日基于国债期货结算价计算的修正反转因子判断𝑻日国债期货的涨跌方向……此前报告中的|𝑐ℎ𝑔𝑡| ∈ [0.03%, 0.5%]"
- 公式（转写）: 本报告未给出完整计算式；仅说明由 T-2 日国债期货结算价计算的"修正反转因子"chg_t，用于判断 T 日涨跌方向；合理区间 |chg_t| ∈ [0.03%, 0.5%]
- 参数: 合理区间 [0.03%, 0.5%]；来源=前作《国债期货的"隔日反转"效应及择时策略构建》（2023-05-13 发布，p11 注4）
- 依赖数据: D1 / 关联基准: R7 / 关联歧义: A1

### [F4] 复合股债跷跷板择时信号 signal_seesaw
- 页码: p13
- 原文: > "𝑠𝑖𝑔𝑛𝑎𝑙𝑠𝑒𝑒𝑠𝑎𝑤 = 𝑠𝑖𝑔𝑛(𝑠𝑖𝑔𝑛𝑎𝑙𝑑𝑎𝑖𝑙𝑦_𝑙𝑜𝑤𝑒𝑟 + 𝑠𝑖𝑔𝑛𝑎𝑙𝑙_𝑠)，其中……𝑠𝑖𝑔𝑛𝑎𝑙𝑙_𝑠 为长短端股债跷跷板信号，𝑠𝑖𝑔𝑛为符号函数"
- 公式（转写）: signal_seesaw = sign(signal_daily_lower + signal_l_s)；由日度低阈值信号与长短端信号简单叠加取符号

- 依赖数据: D1, D2 / 关联基准: R9 / 关联歧义: A2, A3

### [F5] 周内日历效应信号 signal_calendar
- 页码: p15
- 原文: > "𝑠𝑖𝑔𝑛𝑎𝑙𝑑𝑎𝑖𝑙𝑦𝑢𝑝𝑝𝑒𝑟_𝑐𝑎𝑙𝑒𝑛𝑑𝑎𝑟 = 𝑠𝑖𝑔𝑛(𝑠𝑖𝑔𝑛𝑎𝑙𝑑𝑎𝑖𝑙𝑦_𝑢𝑝𝑝𝑒𝑟 + 𝑠𝑖𝑔𝑛𝑎𝑙𝑐𝑎𝑙𝑒𝑛𝑑𝑎𝑟)，其中𝑠𝑖𝑔𝑛𝑎𝑙𝑐𝑎𝑙𝑒𝑛𝑑𝑎𝑟 = 1 𝑖𝑓 𝑑𝑎𝑦 𝑡 𝑖𝑠 𝑇ℎ𝑢𝑟𝑠𝑑𝑎𝑦，0 𝑒𝑙𝑠𝑒"

- 依赖数据: D2, D4 / 关联基准: R11 / 关联歧义: （无）

### [F6] 长短端跷跷板 + 隔日反转策略（改进规则一）
